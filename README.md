
## Description

Notre jeu de Masyu est un jeu de puzzle de logique très amusant. Le but du jeu est de relier des cercles en utilisant des lignes selon des règles spécifiques. Vous allez adorer résoudre ces puzzles !
Plateformes

Notre jeu est développé en Flutter, ce qui signifie qu'il fonctionne sur une variété de plateformes, notamment Windows, Mac, iOS et Android. Nous voulons que tout le monde puisse profiter de notre jeu !

## Comment jouer

Le jeu se compose d'une grille de cases carrées, chacune contenant un cercle noir ou blanc. Les cercles noirs sont des "indices" et les cercles blancs sont des "espaces vides". L'objectif est de relier les cercles avec des lignes pour former une figure fermée.

Les règles sont les suivantes :

1.  Les lignes ne peuvent pas traverser d'autres lignes ni les bords de la grille.
2.  Les lignes doivent passer à travers les cercles en respectant les indications des indices :
    -   Si un cercle noir a un nombre pair de lignes adjacentes, les lignes doivent tourner à angle droit.
    -   Si un cercle noir a un nombre impair de lignes adjacentes, les lignes doivent traverser le cercle en ligne droite.
    -   Si un cercle blanc a deux lignes adjacentes qui passent de part et d'autre du cercle, ces lignes doivent être reliées par une diagonale.
    -   Si un cercle blanc a deux lignes adjacentes qui passent du même côté du cercle, ces lignes ne peuvent pas être reliées.

En utilisant ces règles, vous devez relier tous les cercles avec des lignes pour former une figure fermée. Bonne chance !

## Comment installer

Clonez le dépôt Git : Pour cela, ouvrez une invite de commande ou un terminal sur votre ordinateur et exécutez la commande `git clone` suivie de l'URL du dépôt Git. Par exemple, pour cloner le dépôt Git de notre jeu de Masyu, vous pouvez exécuter la commande suivante :

bash

1.  `git clone https://github.com/rob1juan/masyu_game.git` 
    
2.  Installez les dépendances : Ouvrez le dossier du projet que vous venez de cloner à l'étape précédente dans votre terminal et exécutez la commande `flutter pub get`. Cela installera toutes les dépendances nécessaires pour exécuter l'application.
    
3.  Connectez votre appareil : Si vous souhaitez installer l'application sur votre appareil Android ou iOS, connectez-le à votre ordinateur à l'aide d'un câble USB. Assurez-vous que le mode de développement est activé sur votre appareil.
    
4.  Exécutez l'application : Dans votre terminal, exécutez la commande `flutter run`. Si vous avez plusieurs appareils connectés, vous devrez spécifier le périphérique sur lequel vous souhaitez exécuter l'application. Pour cela, utilisez la commande `flutter devices` pour voir la liste des périphériques disponibles, puis la commande `flutter run -d [device-id]` en remplaçant `[device-id]` par l'ID de votre périphérique.
    
5.  Profitez de l'application : Une fois l'application lancée, vous pouvez en profiter sur votre appareil ! Si vous rencontrez des problèmes ou des erreurs, assurez-vous que toutes les dépendances sont installées et que votre environnement de développement est correctement configuré.
    

J'espère que ces étapes vous aideront à installer l'application Flutter de notre jeu de Masyu sur votre appareil. N'hésitez pas à nous contacter si vous avez des questions ou des problèmes !
